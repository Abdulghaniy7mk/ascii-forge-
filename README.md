# ascii-forge 🔥

> Blazing fast terminal tools. Pure ASCII. No dependencies. Full vibes.

A curated collection of beautiful ASCII animations, dynamic TUIs (Terminal User Interfaces), and terminal-native tools. Everything rendered in glorious 256 colors—built for the terminal first, terminal always.

---

## ✨ Features

- **100% ASCII** — No external rendering libraries. Pure terminal magic.
- **Minimal deps** — Lightweight, portable, modular.
- **Copy-paste ready** — Drop scripts straight into your workflow.
- **Interactive & animated** — Bring life to your terminal.

## 📖 How to Use This Repo

Each tool lives in its own folder with:
- `tool_name.py` (or `.sh`, `.js`) — The main script
- `README.md` — Tool-specific docs
- `demo.txt` or `.gif` — Example output (optional)

## 🎨 Design Philosophy

- **Terminal-first.** No GUI, no web. Full ASCII.
- **Modular.** Pick what you need. No bloat.
- **Copy-paste friendly.** Standalone scripts when possible.
- **Readable code.** Comments. Structure. Vibes.

## 📝 Adding a New Tool

1. Create a folder: `tools/my_cool_tool/`
2. Add your script + a `README.md`
3. Update **this README** with a 2-3 line entry in the Tools section
4. Commit with a clear message
